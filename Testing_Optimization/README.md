# Testing_Optimization

##  통합 테스트

- 회원가입 → 로그인 → 후기 작성 → 평점 반영 흐름 테스트
- 세션 유지 및 로그인 상태 확인
- 후기 작성 제한(중복 방지) 정상 동작 확인
- 상품등록 - > 상품수정,상품삭제,상품검색 흐름 테스트
- 상품 수정,삭제이후 상품 검색이후 검색시 동기화 상태 확인
- 삼품 검색시 사용자가 정해놓은 분류에따라 정렬상태 확인
---

##  최적화

- 불필요한 DB 조회 최소화
- 평점 계산 시 SQL 집계 함수 활용
- DAO 계층 분리로 유지보수성 향상
- 상품 등록 → 조회 → 검색 → 수정 → 삭제 흐름 테스트
- 검색 조건별 결과 확인 및 경계값 테스트(가격범위, 카테고리 없음 등)
- 입력값 검증 테스트(빈값, 음수 가격, 잘못된 카테고리 등)
