# Project_Structure

##  담당 역할: 회원 / 후기 / 평점 관리

###  담당 기능
- 회원가입 및 로그인 기능 구현
- 세션 기반 로그인 유지 처리
- 후기 작성 및 조회 기능
- 사용자 평점 계산 및 출력 기능

---

##  담당 구조 (MVC Model2)

- Controller: UserServlet, ReviewServlet
- Model: User, Review
- DAO: UserDAO, ReviewDAO
- View: login.jsp, register.jsp, review.jsp, mypage.jsp
